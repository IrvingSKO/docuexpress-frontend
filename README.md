## Ejecutar local
1) Instalar dependencias:
   npm install

2) (Opcional) Crear archivo `.env` con:
   VITE_API_URL=http://localhost:3001/api

3) Iniciar:
   npm run dev

## Deploy en Vercel
En Vercel > Settings > Environment Variables:
- Key: VITE_API_URL
- Value: https://docuexpress.onrender.com/api

Luego redeploy.
